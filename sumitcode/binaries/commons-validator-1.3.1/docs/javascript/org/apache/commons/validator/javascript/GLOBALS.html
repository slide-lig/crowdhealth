validateMaxLength

<static> Object validateMaxLength(form)
    A field is considered valid if less than the specified maximum. Fields are not checked if they are disabled. Caution: Using validateMaxLength() on a password field in a login page gives unnecessary information away to hackers. While it only slightly weakens security, we suggest using it only when modifying a password.
    Parameters:
      form - The form validation is taking place on.
